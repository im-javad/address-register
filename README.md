## Setup and Installation

### 1. Clone the Repository:

```bash
git clone https://github.com/im-javad/address-register.git
```

### 2. Go to Project:

```bash
cd next-app
```

### 3. Install Dependencies:

```bash
npm install
```

### 4. Run the Project:

```bash
npm run dev
```

### This will launch the app on `http://localhost:3000/`.
