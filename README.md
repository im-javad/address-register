## Setup and Installation

### 1.Clone the Repository:

```bash
git clone https://github.com/im-javad/address-register.git
cd next-app
```

### 2.Install Dependencies:

```bash
npm install
# or
yarn install
```

### 3.Run the Project:

```bash
npm run dev
```

This will launch the app on `http://localhost:3000/`.
